## Evidence Scope

The final product is the three installable Skills, their bundled scripts and `agents/openai.yaml` metadata, and `templates/论文精读模板.md`. The reports document permission integration, public/local identity, privacy checks, provenance reconstruction, separated clean rewrites, deterministic validation, and final acceptance.

Earlier reports remain in this directory to preserve an honest audit trail. Some record intermediate blockers or decisions that were later superseded. Where an intermediate report conflicts with a final accepted report, the final evidence path above governs the current release state; history is retained rather than rewritten.

ScholarTrace and API experiments were developed on a separate, unmerged branch. They are not part of this directory's submitted product, `main`, or the release candidate.
